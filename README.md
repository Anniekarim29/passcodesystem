# Passcode System

A beautiful and interactive rotary-style passcode input application built with Flutter.

## Features

- Custom interactive rotary dial for inputting digits.
- Intuitive gesture-based user interaction.
- Premium UI aesthetics utilizing glassmorphism effects and modern design principles.
- Clean and responsive view.

## Tech Stack

- **Flutter**: UI toolkit for building modern applications.
- **Dart**: Programming language used to code Flutter apps.
- **Provider**: State management solution.
- **Google Fonts**: Custom typography.

## Getting Started

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run the application with `flutter run`.
